Hexo 的基本使用

一、快速建站

git 与 node 安装不做过多的赘述了

二、主题变更

创建 Hexo 主题非常容易，您只要在 themes 文件夹内，新增一个任意名称的文件夹，并修改 _config.yml 内的 theme 设定，即可切换主题，当然这个_config.yml 指的是当前博客根目录下的文件，不要跟主题下的混淆了，下面可能是主题的基本结构

  1. 安装主题

    打开 主题预览官网 找到合适的主题，找到它们的github地址，使用如下命令进行克隆

  2. 切换主题

    修改 Hexo 根目录下的 _config.yml 文件

  3. 启动 Hexo 服务器, 进行查看是否切换成功
